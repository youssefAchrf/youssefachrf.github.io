Graduation Project: EVCC (Electric Vehicle Charge Controller)

An embedded system for electric vehicles that manages AC/DC charging, estimates battery states, and enables ISO 15118-based Vehicle-to-Grid (V2G) communication with a 48V lithium battery.

  • Designed and validated State of Charge (SOC) and State of Health (SOH) estimation algorithms in MATLAB/Simulink using Extended Kalman Filter (EKF) and Coulomb Counting methods.
  • Developed SOC estimation algorithms in Embedded C specifically for the NXP microcontroller.
  • Debugged the final firmware on the hardware using a Lauterbach debugger, verifying the real-time performance of the SOC driver.

Marble Defect Detection (Machine Learning)

Collaborated with a marble manufacturing company to develop an automated defect detection system (cracks, breakage) for their output. Designed and implemented a YOLO model for real-time object detection and localization of various marble defects.
